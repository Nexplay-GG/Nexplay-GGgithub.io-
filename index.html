🛠 Aggiungi un Gioco

Hai creato un gioco HTML/Canvas? Aggiungilo in 3 secondi. Basta modificare l'array GAMES nello script.

01
Prepara il file
Il tuo gioco deve essere un file HTML completo e autonomo (come Flappy Bee).
02
Aggiungi all'array
Apri il file del portale e aggiungi un oggetto nel registro GAMES con id, nome, emoji e HTML.
03
Deploy & Gioca
Salva, ricarica — la card appare automaticamente nella griglia, cliccabile e pronta.
// ── Aggiungi il tuo gioco qui sotto ──
const GAMES = [
  {
    id: 'flappybee',
    name: 'Flappy Bee',
    emoji: '🐞',
    genre: 'Arcade',
    rating: '4.9',
    badge: 'new', // 'new' | 'hot' | 'coming' | null
    html: FLAPPY_BEE_HTML, // variabile con l'HTML del gioco
  },
  // ── Copia questo blocco per il prossimo gioco ──
  {
    id: 'ilmiogioco',
    name: 'Il Mio Gioco',
    emoji: '🚀',
    genre: 'Action',
    rating: '5.0',
    badge: 'hot',
    html: IL_MIO_GIOCO_HTML,
  },
];
